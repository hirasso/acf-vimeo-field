# Installation

This plugin should only be installed in a project that is fully managed by composer, like [roots/bedrock](https://github.com/roots/bedrock) or [vinkla/wordplate](https://github.com/vinkla/wordplate).

```shell
# install
composer require hirasso/acf-vimeo-field
# activate using wp cli
wp plugin activate acf-vimeo-field
```
